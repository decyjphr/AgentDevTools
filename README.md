# AgentDevTools

[![Deploy Documentation](https://github.com/decyjphr/AgentDevTools/actions/workflows/docs.yml/badge.svg)](https://github.com/decyjphr/AgentDevTools/actions/workflows/docs.yml)

A curated documentation hub for tools, frameworks, and libraries used to build AI Agents.

📖 **[View the Documentation Site →](https://decyjphr.github.io/AgentDevTools/)**

## Covered Tools

| Tool | Description |
|------|-------------|
| [LangChain](https://decyjphr.github.io/AgentDevTools/tools/langchain/) | General-purpose LLM chain & agent framework |
| [LlamaIndex](https://decyjphr.github.io/AgentDevTools/tools/llamaindex/) | RAG and data-augmented agent framework |
| [AutoGen](https://decyjphr.github.io/AgentDevTools/tools/autogen/) | Multi-agent conversation framework by Microsoft |
| [CrewAI](https://decyjphr.github.io/AgentDevTools/tools/crewai/) | Role-based multi-agent orchestration |
| [Semantic Kernel](https://decyjphr.github.io/AgentDevTools/tools/semantic-kernel/) | Enterprise AI SDK by Microsoft |

## Local Development

```bash
pip install mkdocs-material
mkdocs serve
```

Open [http://localhost:8000](http://localhost:8000) to preview the documentation site.
